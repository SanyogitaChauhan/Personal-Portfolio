Project Overview

A Sudoku game project involves creating a digital version of the popular puzzle game where the objective is to fill a 9x9 grid with numbers so that each column, each row, and each of the nine 3x3 subgrids contain all of the digits from 1 to 9, without repetition.

The main interface is a 9x9 grid, consisting of 81 cells divided into nine 3x3 subgrids. Offer varying difficulty levels like Easy, Medium, Hard, and Expert. Each level would have different levels of pre-filled numbers to adjust difficulty.

Designed and implemented a backtracking algorithm to generate and solve Sudoku puzzle.Developed an interactive GUI which enables users to input numbers.
